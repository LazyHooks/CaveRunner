github.com/lazyhooks
Scripting Language : GML
Graphics Engine GMStudio 2G graphics.
Pixellated Platformer Game.

Controls:
Keyboard : arrow keys.
Gamepad: LS to move left, right ,and down. RB to jump.

General:
Jumping on enemies kills them. 
Every new level is a checkpoint. 
Highscores get stored. 
Includes Boss.
